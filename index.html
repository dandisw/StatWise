1. Penyiapan Variabel (Variable View)

Sebelum memasukkan data, definisikan variabel Anda. Ubah nama variabel, atur tipe data (Kategori untuk perlakuan, Numerik untuk data hasil). Isi kolom Values dengan format 1=Kontrol, 2=Pakan_A untuk memudahkan pelabelan grafik.

2. Memasukkan Data (Data View)

Anda dapat mengetik langsung, atau melakukan Copy-Paste dari Microsoft Excel/Google Sheets. Klik sel pada tabel web, lalu tekan Ctrl + V.

3. Analisis & Output

Gunakan panel di sebelah kiri untuk menjalankan uji statistik. Semua hasil analisis akan ditumpuk secara berurutan di Laporan Analisis (sebelah kanan). Aplikasi menyediakan interpretasi kesimpulan teks secara otomatis.

4. Menyimpan & Mengekspor

Klik "Ekspor PDF" pada panel hasil. Sistem otomatis merapikan tata letak, menyembunyikan menu, dan menyajikan laporan putih bersih yang siap dicetak atau disimpan sebagai PDF untuk dilampirkan pada skripsi/jurnal Anda.

Aplikasi ini ditenagai oleh mesin komputasi matematis luring yang diselaraskan dengan standar pengujian perangkat lunak profesional (IBM SPSS & R CRAN). Berikut adalah referensi akademiknya:

  • Uji Normalitas (Shapiro-Wilk): Berdasarkan Royston, P. (1992). Approximating the Shapiro-Wilk W-test for non-normality. Dilengkapi koreksi Lilliefors (Dallal-Wilkinson, 1986) untuk uji Kolmogorov-Smirnov.
  • Uji Homogenitas (Levene): Standar kokoh menolak outlier menggunakan berbasis median dari Brown, M. B., & Forsythe, A. B. (1974). Terdapat opsi Bartlett's Test parametrik.
  • Notasi Huruf Signifikansi (a, b, c): Menggunakan algoritma Sweep CLD berstandar jurnal berdasarkan Piepho, H. P. (2004). An algorithm for a letter-based representation of all-pairwise comparisons.
  • Sampel Kecil Non-Parametrik: Menerapkan distribusi permutasi eksak (Exact P-Value) secara otomatis jika ukuran sampel (N) di bawah 20 untuk meminimalisasi galat asimtotik Z-Score.
  • GLM ANOVA Faktorial: Menerapkan eliminasi inversi aljabar matriks (Gauss-Jordan) untuk eksekusi Type III Sum of Squares pada desain yang tidak seimbang (*Unbalanced Design*).
Versi 10.0 (Current) - Mobile & Documentation Update
Penambahan tata letak responsif penuh untuk ponsel/tablet, integrasi dokumentasi rujukan, dan restrukturisasi antarmuka pengguna tanpa memutus komputasi statik.
Versi 9.0 - Absolute SPSS/R Parity
Penutupan celah statistik (*vulnerability*): Penerapan matriks GLM Type III SS, penanganan Missing Data, K-S Lilliefors *Approximation*, dan *Exact* Mann-Whitney untuk data N kecil.
Versi 7.0 & 8.0 - Transformasi & Ekstraksi Ekstensif
Modul transformasi logaritma, akar, dan arkus sinus ditambahkan. Menggabungkan uji t-Student, Welch, Korelasi Pearson/Spearman, Regresi Linear, Boxplot interaktif, Dunn's test, dan tombol ekstrak tabel data CSV/TSV.
Versi 5.0 & 6.0 - Master Reporting
Lahirnya sistem panel laporan kumulatif (*append* uji statistik tiada batas) dengan cetak luring PDF native serta interpretasi kesimpulan teks bahasa ilmiah secara otomatis.
Versi 1.0 hingga 4.0 - Fondasi & Presisi Matematis
Pengembangan arsitektur Single Page Application dengan input matriks Excel. Pengembangan engine Incomplete Beta & Log-Gamma untuk resolusi *p-value* tingkat ekstrim. Integrasi otomatis Notasi Uji Lanjut Piepho (2004).
